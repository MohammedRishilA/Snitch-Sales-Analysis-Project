# Snitch-Sales-Analysis-Project
Sales data analysis project for a fashion brand, focusing on cleaning, exploring, and visualizing sales data to uncover trends, top products, and customer behavior. Built using Python (Pandas, NumPy) and Power BI to generate actionable insights for improving business performance and decision-making.
# 📊 Sales Data Analysis Project

## 🔹 Project Overview
This project analyzes sales data using Power BI and Python to generate insights and support business decisions.

## 🔹 Problem Statement
The goal is to understand sales performance, identify trends, and improve profitability.

## 🔹 Tools & Technologies
- Power BI
- Python (Pandas, Matplotlib)
- Excel / CSV Dataset

## 🔹 Project Steps
1. Data Cleaning using Python
2. Data Analysis
3. Data Visualization
4. Dashboard Creation in Power BI

## 🔹 Key Insights
- Top-selling products contribute 40% of revenue
- Sales increased during festive months
- Profit margin varies by region
